# Portfolio-Manager-CrewAi

This repository demonstrates the step-by-step development of an **AI Portfolio Manager** using CrewAI.  

The project evolves in phases to showcase how agentic systems can scale from a **simple single-agent setup** to a **multi-agent, tool-driven, RAG-powered system** with metrics and MCP integration.  

---

## Phases

1. **Phase 1** – Single Agent, No Tools  
2. **Phase 2** – Multiple Agents + Tools  
3. **Phase 3** – Task-Specific Agents  
4. **Phase 4** – Metrics & Benchmarks  
5. **Phase 5** – RAG Implementation  
6. **Phase 6** – MCP Server Integration  

---
