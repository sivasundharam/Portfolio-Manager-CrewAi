# Portfolio-Manager-CrewAi Phase 1.1

---

## Phase 1.1: Single Agent with Tools

### Objective
Build a minimal **portfolio manager agent** that can:
- Analyze current holdings (CSV/JSON input provided)
- Provide a simple portfolio summary (e.g., diversification, total value, risk level)
- Give a basic text-only recommendation

**Tools**
- stock_price_tools
- portfolio_analysis_tool
- risk_calculator_tool

This will act as the **baseline** for later phases.

---

## Tech Stack
- Python 3.10+
- CrewAI
- LangChain
- OpenAI LLM
